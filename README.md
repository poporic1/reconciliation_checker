# Data Reconciliation Tool

Сравнения данных между двумя таблицами (или CSV файлами).  
Помогает быстро проверить, совпадают ли данные, есть ли пропуски, дубли и различия.

---

## Возможности

Скрипт выполняет несколько проверок:

- сравнение количества строк  
- поиск дубликатов по ключам  
- поиск отсутствующих ключей (в source и target)  
- сравнение значений по выбранным колонкам  

В конце формируется итоговый отчет.

---

## Как это работает

1. Читается конфигурационный файл  
2. Загружаются два CSV файла  
3. Выполняются проверки:
   - количество строк  
   - дубли  
   - отсутствующие ключи  
   - различия в значениях  
4. Формируется итоговая сводка  
5. Результат сохраняется в файл  

---

## Конфигурация

Файл: `config/reconciliation_config.json`

```
  json
{
  "source_file": "data/source_orders.csv",
  "target_file": "data/target_orders.csv",
  "key_columns": ["id"],
  "compare_columns": ["amount", "status"]
}
```

---

## Описание параметров:

source_file — исходный файл
target_file — файл для сравнения
key_columns — ключевые поля (по ним идет сравнение строк)
compare_columns — колонки, значения которых нужно сравнить

---

## Запуск:

1. Установить зависимости
pip install pandas
2. Подготовить данные
Положить CSV файлы в папку data/
3. Настроить конфиг
Отредактировать файл:
config/reconciliation_config.json
4. Запустить скрипт
python main.py

---

## Результат:

После запуска создается файл:
output/summary_result.csv
В нем содержится:
- количество строк в source и target
- совпадают ли они
- количество отсутствующих ключей
- количество различий в значениях
- время выполнения проверки
